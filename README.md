# Day-18-Claude-AI-challenge-
Day 18: Custom Skill - Brain Dump to Action Planner
Built a reusable Claude Custom Skill (brain-dump-action-planner) that converts unstructured notes, meeting transcripts, and voice memos into structured HTML dashboards.

- Outputs: Summary, Key Takeaways, Action Items table, Open Questions, Risks/Blockers, Conflicts, Additional Notes
- Transcript Mode adds speaker-level attribution: Speaker Summary, Decisions by Speaker, Action Items by Speaker
- Strict rule: never invents or assumes missing information — displays "Not specified" instead
- Conflicts are flagged, never auto-resolved
- Tested on 2 note formats: standard meeting notes and multi-speaker transcript
